## Введение
В рамках данного курсового проекта была разработана файловая система библиотеки в виде консольного приложения. В процессе работы использовались знания и навыки, полученные на лекциях и отработанные на практике.

## Постановка задачи
Целью проекта было создание картотеки с возможностями изменения базы и её сортировки.

## Реализация
Работа выполнена с использованием следующих компонентов:

- **menu.cpp**: реализация меню, вывод списка возможных действий.
- **book.cpp**: создание жанров книг, вывод книг, ввод данных о книге через консоль.
- **library.cpp**: подключение файлов формата .txt с книгами, реализация удаления картотеки и отдельных книг, сохранение картотеки в файл, чтение книг из файла и сохранение в картотеку.
- **sort.cpp**: сортировка картотеки по названию и по автору.

## Тестирование
Произведено тестирование функционала, включающее в себя:
- Проверку работы меню и выбор опций.
- Ввод и удаление книг.
- Вывод картотеки в консоль.
- Запись текущей картотеки в файл.
- Выход из программы.

## Заключение
В процессе выполнения проекта были получены навыки разработки консольных программ на основе изученных знаний и реализации задач, поставленных передо мной в рамках курса.
